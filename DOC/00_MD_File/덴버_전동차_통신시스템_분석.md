> 작성일: 2026-04-02

## 1. 프로젝트 개요

미국 콜로라도주 덴버 RTD 경전철 전동차에 탑재되는 **승객 안내 및 통신 시스템(PIS/Communication System)** 개발·납품 프로젝트.  
현대로템이 차량 제작을 담당하고, 우진산전(JDS)이 통신 시스템 서브시스템을 공급하는 구조.

### 1.1 차량 구성
- A Car / B Car 편성 (2량 1편성 기준, 중련 운행 지원)
- 총 50량 규모 (예비품 포함)

### 1.2 프로젝트 타임라인 (폴더 기반 추정)
| 시기 | 주요 활동 |
|------|----------|

| 2012.01 | SEPTA 참고 자료 검토, 사양 확정 |
| 2012.03 | 현차 조치 방안 수립 (접지, 커넥터, 트랜스포머) |

## 2. 시스템 구성 장치

### 2.1 주요 장치 목록

| 장치 약어 | 장치명 | 기능 |
|-----------|--------|------|
| **CRA** | Central Rack Assembly | 중앙 처리 장치 (메인 컨트롤러) |
| **MDS** | Master Display System (CDT) | 마스터 디스플레이 / 안내 표시 장치 |
| **PID** | Passenger Information Display | 객실 승객 안내 표시기 |
| **PEI** | Passenger Emergency Intercom | 승객 비상 인터콤 |
| **ISP** | Interior Speaker Panel | 객실 내 스피커 패널 |
| **LIC** | Line Intercom Controller | 라인 인터콤 제어기 |
| **CCP** | Cab Control Panel | 운전실 제어 패널 |
| **CNCS** | Communication Network Control System | 통신 네트워크 제어 시스템 |
| **ANSM** | Automatic Next Stop Module | 자동 다음 정거장 안내 모듈 |
| **DPO** | Driver's Panel Operator | 운전자 패널 조작기 |
| **SDI** | Serial Data Interface | 직렬 데이터 인터페이스 |
| **RMM** | Repair & Maintenance Manual | 정비 매뉴얼 (문서) |

### 2.2 통신 프로토콜 및 네트워크

| 기술 | 용도 |
|------|------|
| **LON Works** | 차량 내 제어 네트워크 (Echelon 기반) |
| **RS-485** | CNCS 직렬 통신 |
| **VoIP** | 음성 통신 (AudioCodes 프로세서 기반) |
| **WiMAX** | 무선 데이터 통신 (차량-지상 간) |
| **FTP** | 파일 전송 (데이터 다운로드/업로드) |
| **CCP 통신** | RTS 제어 기반 Cab 간 통신 |
| **CI (Consist Interface)** | 중련 편성 간 인터페이스 |

## 3. 하드웨어 설계

### 3.1 회로도 (PCB 보드)
- **LIC-C1** : LIC 메인 보드 Rev.1
- **LIC-C2** : LIC 보드 Rev.2
- **LIC-C3 / C3A** : LIC 보드 Rev.3 (DXF, GERBER, PCB 파일 포함)
- **APEM Switch** : 운전실 조작 스위치 회로

### 3.2 주요 부품 (데이터시트 기반)
| 카테고리 | 부품 |
|----------|------|
| CPU | TMS320VC33 (DSP), Qseven 보드 |
| 통신 IC | XR16L784 (UART), MAX3535E (RS-485) |
| CPLD | XC95288XL (Xilinx) |
| VoIP | AudioCodes VoIP Processor, CFM53281 |
| 스피커 | Fly Installed Sound 시리즈 |
| 마이크 | AKG PS3F, Gooseneck MIC (MX405/410) |
| 스위치 | APEM Switch, EAO 시리즈, E-Switch |
| 커넥터 | Souriau MSG, Glenair (넥서스), ITH 시리즈 |
| 전원 | MZH30xx 시리즈 DC-DC |
| 네트워크 | MOXA TN-5508/5516 산업용 스위치 |
| 기타 | ULN2803 (드라이버), 슈퍼캐패시터 |

### 3.3 도면 관리
- 덴버 승인도 (ROTEM 승인)
- 차량 배선도 (A Car: RGA10001, B Car: RGA10002)
- 결선도 (2012.09 수정본)

## 9. 핵심 요약

- **프로젝트 성격**: 미국 덴버 RTD 경전철 전동차 통신 시스템 (PIS/PA/Intercom/VoIP)
- **시스템 규모**: CRA, MDS, PID, PEI, ISP, LIC, CCP, CNCS, ANSM 등 10종 이상 장치
- **통신 기술**: LON Works + RS-485 + VoIP + WiMAX 복합 네트워크
- **핵심 HW**: TMS320VC33 DSP, Xilinx CPLD, AudioCodes VoIP, MOXA 산업용 스위치
- **주요 이슈**: CNCS 발열, 자동 방송 불량, RS-485 통신 불안정, PID 전원/영상 불량 등 30건 이상
- **문서 체계**: CMMI 기반 SW 공학 문서 + IPC/RMM/HMM 정비 문서 + EMI/EMC 인증
- **협력사**: RF_COM, 아이텍, 엑스바이더, 넥서스 등 10개 이상 업체 참여
